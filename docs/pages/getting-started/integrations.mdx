# 集成方案

以下是 `@auth/*` 和 `@next-auth/*` 范畴下以及 `next-auth` 相关的计划中和已发布的集成状态，同时也包含社区创建和维护的集成方案。标记为"计划中"的集成项目我们非常期待您的协助！详见下方[需要帮助](#help-needed)章节。

<Callout>
  您的框架尚未获得支持？您可以通过[遵循本指南](/guides/creating-a-framework-integration)创建框架集成来轻松贡献代码。
</Callout>

<Callout type="info">
  请注意，为防止包导入的破坏性变更，`next-auth` 是唯一未遵循 `@auth/*`
  命名规范的框架包。该库最初以 `next-auth`
  命名，但[经过多年发展](/contributors#history)已成为框架无关方案。
</Callout>

框架与数据库集成均基于[Auth.js核心库](/reference/core)实现。大多数情况下您无需直接操作该包，因其主要面向库作者设计。

| 框架                                  | 状态                                                                                |
| ------------------------------------- | ----------------------------------------------------------------------------------- |
| NextAuth.js (`next-auth`)             | [v5 (测试版)](https://nextjs.authjs.dev), [v4 (维护模式)](https://next-auth.js.org) |
| Astro Auth (`@auth/astro`)            | [开放PR](https://github.com/nextauthjs/next-auth/pull/9856)                         |
| Express Auth (`@auth/express`)        | [实验性发布](https://express.authjs.dev)                                            |
| Fastify Auth (`@auth/fastify`)        | [开放PR](https://github.com/nextauthjs/next-auth/pull/9587)                         |
| Nuxt Auth (`@auth/nuxt`)              | [开放PR](https://github.com/nextauthjs/next-auth/pull/10684)                        |
| Qwik Auth (`@auth/qwik`)              | [已发布](https://qwik.authjs.dev)                                                   |
| Remix Auth (`@auth/remix`)            | [开放PR](https://github.com/nextauthjs/next-auth/pull/6767)                         |
| SolidStart Auth (`@auth/solid-start`) | [实验性发布](https://authjs.dev/reference/solid-start)                              |
| SvelteKit Auth (`@auth/sveltekit`)    | [实验性发布](https://sveltekit.authjs.dev)                                          |

| 数据库                 | 状态                                             |
| ---------------------- | ------------------------------------------------ |
| `@auth/*-adapter`      | 已发布。完全兼容 `next-auth` 和所有 `@auth/*` 库 |
| `@next-auth/*-adapter` | 已停止维护。请升级至 `@auth/*-adapter`           |

社区已为各类框架和库发布了一些优秀的集成/客户端包。我们未来希望将部分包转为官方维护，如果您是相关包的维护者并有合作意向，欢迎随时联系我们！

### 需要协助

如果您是使用 `@auth/core` 的软件包维护者，欢迎[联系 Balázs](https://twitter.com/balazsorban44) 或发送邮件至 [info@authjs.dev](mailto:info@authjs.dev)，若有意合作将其发展为官方维护的软件包。如果您有兴趣为您喜爱的框架添加 `@auth/core` 支持，我们非常期待您的来信！
