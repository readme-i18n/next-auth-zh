# Fauna 适配器

## 相关资源

- [Fauna 文档](https://docs.fauna.com/fauna/current/)

## 配置

### 安装

```bash npm2yarn
npm install @auth/fauna-adapter fauna
```

### 环境变量

```sh
AUTH_FAUNA_CLIENT=http://localhost:8443
AUTH_FAUNA_SECRET=abc123
```

### 迁移至 v2 版本

在 `@auth/adapter-fauna@2.0.0` 中，我们根据 Fauna 最佳实践将集合名称改为大写命名模式。若要从 v1 迁移，您需要重命名集合以匹配新命名方案。此外，我们已将索引重命名为方法式索引名称（例如 `account_by_user_id` 改为 `Account.byUserId`）。有关迁移 Fauna 架构的更多信息，请参阅其迁移指南[此处](https://docs.fauna.com/fauna/current/migration)

#### 架构

在 Fauna 仪表板的 `Shell` 标签页中运行以下命令来设置相应的集合和索引。

```ts filename="authjs-fauna-adapter-schema.fql"
Collection.create({
  name: "Account",
  indexes: {
    byUserId: {
      terms: [
        { field: "userId" }
      ]
    },
    byProviderAndProviderAccountId: {
      terms [
        { field: "provider" },
        { field: "providerAccountId" }
      ]
    },
  }
})
Collection.create({
  name: "Session",
  constraints: [
    {
      unique: ["sessionToken"],
      status: "active",
    }
  ],
  indexes: {
    bySessionToken: {
      terms: [
        { field: "sessionToken" }
      ]
    },
    byUserId: {
      terms [
        { field: "userId" }
      ]
    },
  }
})
Collection.create({
  name: "User",
  constraints: [
    {
      unique: ["email"],
      status: "active",
    }
  ],
  indexes: {
    byEmail: {
      terms [
        { field: "email" }
      ]
    },
  }
})
Collection.create({
  name: "VerificationToken",
  indexes: {
    byIdentifierAndToken: {
      terms [
        { field: "identifier" },
        { field: "token" }
      ]
    },
  }
})
```

#### 自定义集合名称

如需使用自定义集合名称，可将其作为选项传递给适配器，如下所示：

```js
FaunaAdapter(client, {
  collectionNames: {
    user: "CustomUser",
    account: "CustomAccount",
    session: "CustomSession",
    verificationToken: "CustomVerificationToken",
  },
})
```

请确保传递给提供程序的集合名称与 Fauna 数据库中的集合名称一致。
