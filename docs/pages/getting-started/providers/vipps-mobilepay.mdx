# Vipps MobilePay 提供商

[Vipps MobilePay](https://vippsmobilepay.com/) 是一款在挪威、瑞典、丹麦和芬兰广泛使用的移动支付应用。该品牌分为两部分：挪威和瑞典使用 Vipps，丹麦和芬兰使用 MobilePay，但两个品牌/应用都使用相同的 API。

## 相关资源

- [Vipps MobilePay 登录文档](https://developer.vippsmobilepay.com/docs/APIs/login-api/)
- [官方 Vipps MobilePay 按钮](https://developer.vippsmobilepay.com/docs/knowledge-base/design-guidelines/buttons/)
- [Vipps MobilePay 公共测试发现端点](https://apitest.vipps.no/access-management-1.0/access/.well-known/openid-configuration)
- [Vipps MobilePay 公共生产发现端点](https://api.vipps.no/access-management-1.0/access/.well-known/openid-configuration)

### 环境变量

```
AUTH_VIPPS_ID
AUTH_VIPPS_SECRET
```

### 测试 API

要使用测试模式，您需要使用测试 API 端点覆盖 issuer。

```
Vipps({ issuer: "https://apitest.vipps.no/access-management-1.0/access/" })
```
