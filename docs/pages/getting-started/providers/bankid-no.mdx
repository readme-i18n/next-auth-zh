# BankID 挪威服务提供商

[BankID Norge](https://bankid.no) 是挪威广泛使用的登录方式，被银行、政府机构和其他组织采用。该提供商允许用户使用挪威 BankID 进行登录。

## 相关资源

- [BankID 挪威文档](https://confluence.bankidnorge.no/confluence/pdoidcl)
- [BankID 测试指南](https://developer.bankid.no/bankid-with-biometrics/testing)
- [BankID 公共测试发现端点](https://auth.current.bankid.no/auth/realms/current/.well-known/openid-configuration)

### 配置

<Code>
  <Code.Next>

```ts filename="/auth.ts"
import NextAuth from "next-auth"
import BankIDNorway from "next-auth/providers/bankid-no"

export const { handlers, auth, signIn, signOut } = NextAuth({
  providers: [BankIDNorway],
})
```

  </Code.Next>

```ts filename="/src/routes/plugin@auth.ts"
import { QwikAuth$ } from "@auth/qwik"

export const { onRequest, useSession, useSignIn, useSignOut } = QwikAuth$(
```
