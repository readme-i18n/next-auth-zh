### 注意事项

您需要选择 AWS 区域才能进入 Cognito 仪表盘。

<Callout type="info">
  颁发者是一个 URL，格式如下：`https://cognito-idp.{region}
  .amazonaws.com/{PoolId}`，其中 `PoolId` 来自 Cognito 的通用设置，
  注意不要与应用客户端 ID 混淆。
</Callout>

<Callout type="info">
  在配置这些设置前，您必须先设置 Amazon Cognito 托管域名。 该设置位于
  `应用客户端/编辑托管 UI` 中。
</Callout>

<Callout type="warning">
  请确保选择所有适用的客户端设置，否则 OAuth 流程将无法工作。
</Callout>
