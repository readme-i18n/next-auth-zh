### 注意事项

- 创建应用程序时，请确保选择 `Authentication & API Access` 作为连接类型。同时确保勾选 `publicData` 作用域。

- 如果使用 JWT 进行会话管理，可以将 `CharacterID` 添加到 JWT 和会话中。例如：

```ts
const AuthConfig = {
  callbacks: {
    jwt({ token, profile }) {
      if (profile) {
        token.characterId = profile.CharacterID
      }
      return token
    },
    session({ session, token }) {
      session.user.characterId = token.characterId
      return session
    },
  },
}
```
