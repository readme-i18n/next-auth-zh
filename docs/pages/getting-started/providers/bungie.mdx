```
AUTH_BUNGIE_ID

AUTH_BUNGIE_API_KEY
```

### 配置

访问 https://www.bungie.net/en/Application 并填写必要信息：

- 应用名称
- 应用状态
- 网站
- OAuth 客户端类型
  - 保密型
- 回调 URL
  - https://localhost:3000/api/auth/callback/bungie
- 权限范围
  - `访问您的 Bungie.net 通知、会员资格及近期 Bungie.Net 论坛活动等项目`
- 来源标头

### 注意事项

- Bungie 要求所有客户端必须使用 **https**
- Bungie 不允许使用主机名 `localhost`，因此在本地开发时必须改用 `127.0.0.1`
