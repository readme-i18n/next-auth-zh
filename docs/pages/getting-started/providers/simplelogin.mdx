## 注意事项

### 授权回调 URI

使用的"授权回调 URI"必须包含完整域名并以回调路径结尾。默认情况下，SimpleLogin 已将所有的 `http[s]://localhost:*` 地址加入白名单以方便本地开发。例如：

- 生产环境：`https://{YOUR_DOMAIN}/api/auth/callback/simplelogin`
- 开发环境：默认情况下 **localhost** 已被加入白名单。

出于安全考虑，**授权回调 URI** 必须使用 **HTTPS**（`localhost` 除外）。
