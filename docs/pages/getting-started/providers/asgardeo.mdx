### 配置

按照以下步骤操作：

1. 登录 [Asgardeo 控制台](https://console.asgardeo.io)
2. 进入"Applications"选项卡（更多信息请参考[此处](https://wso2.com/asgardeo/docs/guides/applications/register-oidc-web-app/)）
3. 注册一个基于标准的 OpenID Connect 应用程序
4. 添加**回调 URL**：`http://localhost:3000/api/auth/callback/asgardeo`（开发环境）和 `https://{YOUR_DOMAIN}.com/api/auth/callback/asgardeo`（生产环境）
5. 注册应用后，进入"Protocol"选项卡
6. 勾选 `code` 作为授权类型
7. 添加"Authorized redirect URLs"和"Allowed origins fields"字段
8. 在控制台中设置 Email、First Name 和 Photo URL 为用户必填属性

然后将 ClientID、ClientSecret 和 Issuer 值添加到您的环境变量中。
