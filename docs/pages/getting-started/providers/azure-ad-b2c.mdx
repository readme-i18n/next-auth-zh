### 租户设置

基础配置将 Azure AD B2C 设置为返回 ID 令牌。这应在执行高级配置之前作为先决条件完成。在租户设置中，请确保在"用户属性和令牌声明"阶段进行以下设置：

- 收集属性：
  - 电子邮件地址
  - 显示名称
  - 名字
  - 姓氏
- 返回声明：
  - 电子邮件地址
  - 显示名称
  - 名字
  - 姓氏
  - 身份提供商
  - 身份提供商访问令牌
  - 用户对象 ID
