Auth.js 遵循负责任披露原则。如发现可能影响 Auth.js 使用站点安全的严重问题，请直接联系我们进行报告。

若您就严重问题联系我们：

<Steps>

### 回复时效

我们将在72小时内尽快给予您回复。

### 修复发布

我们力争在30天内发布修复方案。

### 问题披露

在发布修复方案后（经您同意的情况下将注明您的贡献），我们将公开披露该问题。

### 90天期限

若90天后仍未发布修复方案，我们将公开披露该问题。

</Steps>

报告问题的最佳方式是通过电子邮件联系我们：info@balazsorban.com、hi@thvu.dev、yo@ndo.dev 和 hi@ubbe.dev，或者提交公开问题——**不披露任何敏感细节**——请求相关人员通过您偏好的方式与您联系以获取更多详情。

<Callout type="info">
  对于不太严重的问题（例如不受支持流程的 RFC
  合规性或未来可能引发问题的潜在问题），适合将其作为错误报告或功能请求公开，或提出问题以围绕它们展开讨论。
</Callout>
